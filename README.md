# scruby-instructions

Single-page gameplay instructions for the [Scruby Companion](https://www.curseforge.com/hytale/mods/scruby-companion) mod, used as the "Gameplay Instructions Document" for the Hytale New Worlds Contest submission.

Live page: published via GitHub Pages from the `main` branch root.

Vanilla HTML/CSS, no frameworks, no JavaScript, no tracking.
